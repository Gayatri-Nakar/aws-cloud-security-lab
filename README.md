# AWS Cloud Security Lab

## Overview

This project documents the design and implementation of a production-inspired AWS cloud security lab built entirely with native AWS services.

The objective is to gain hands-on experience with AWS networking, identity and access management, monitoring, threat detection, incident investigation, and security best practices while documenting the engineering process from start to finish.

## Project Goals

- Learn core AWS services through hands-on implementation
- Build a secure cloud environment from the ground up
- Configure monitoring and logging using AWS-native services
- Investigate real-world attack activity
- Document architecture, lessons learned, and design decisions

## Planned Services

- IAM
- VPC
- EC2
- Security Groups
- CloudTrail
- CloudWatch
- VPC Flow Logs
- GuardDuty
- Security Hub
- Amazon Inspector
